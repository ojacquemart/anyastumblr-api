# hfrAsTumblr


Play Framework Scala / AngularJS / MongoDB to tumblerize some sites with funny things.

http://hfrastumblr.herokuapp.com/

## Stack

* Play framework, with Scala
* MongoDB, with Reactive Mongo
* AngularJS, with Yeoman


### Install

- git clone %REPO%
- play
- idea
- npm install
- bower install
- run
